void nana::prop::grid::Set ( vector_t &  v)

Add properties vector vector.

Parameters
[in]vvector of properties

The properties in this vector will be used to initialize the grid. When user edits property values in the grid the property values in this vector will be updated.

A pointer to the vector is stored, so the calling code must ensure that the vector does not go out of scope before the grid does
